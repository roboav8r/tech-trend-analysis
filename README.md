# Tech Trend Analysis [WIP]
A repo that accepts a list of websites and .pdf documents, parses them using an LLM to determine relevant past, present and future trends, and then generates a report for the user.

![Example wordcloud output](future_concepts_wordcloud.png "Example Wordcloud Output")

# Setup
## Create mamba environment
mamba env create -f env.yml

# Usage
mamba activate tech_trend_analysis
